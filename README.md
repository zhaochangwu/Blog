# Blog

## Hexo

快速、简洁且高效的博客框架

- [官网](https://hexo.io/zh-cn/)
- [GitHub](https://github.com/hexojs/hexo)

### 命令

``` shell
$ hexo new [layout] <title>
$ hexo publish [layout] <filename>
$ hexo server
$ hexo deploy
```

### 正常部署(备忘)

``` shell
$ hexo generate -d
```

### Theme
[Next](https://github.com/theme-next/hexo-theme-next)

$ cd hexo
